# 合约杠杆与保证金参数工具

本地服务端页面，用于查询 Gate USDT 永续合约的基础参数、风险限额梯度，并在档位超过 10 档时按相邻两档合并为一档进行精简。

## 启动

```bash
python3 server.py
```

浏览器打开：

```text
http://127.0.0.1:8765
```

服务器直接通过 IP 访问时启动：

```bash
HOST=0.0.0.0 PORT=8765 python3 server.py
```

访问：

```text
http://服务器IP:8765
```

## 当前功能

- 交易所选择：当前支持 `gate.io`
- 合约交易对列表：拉取 Gate 当前 USDT 永续合约
- 合约搜索和单选
- 合约交易对介绍
- 合约基础参数
- 风险限额 / 杠杆 / 保证金梯度
- 风险限额 / 杠杆 / 保证金梯度（精简）
- 选中合约后，标记价格、指数价格、合约基础参数和梯度表每 5 秒实时刷新

## 合约交易对介绍

介绍文案来自：

```text
data/coin_overviews.json
```

文件 key 使用合约基础币种，例如 `ACN_USDT` 对应 `ACN`。

如果某个币种尚未维护介绍，页面显示“暂无币种概况。”，不会自动拼接不准确的介绍。

## 精简规则

当原始梯度档位数大于 10 时，每相邻两档合并为一个精简档位，并采用该组后一档的风险限额、杠杆和保证金率作为精简档参数。

## 服务器部署建议

建议将项目放在：

```text
/opt/apps/futures-risk-tool
```

项目内提供了 systemd 示例：

```text
deploy/futures-risk-tool.service
```

使用前需要把文件里的 `YOUR_SERVER_USER` 改成服务器用户名。
